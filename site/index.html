The inversion

Brainstorming · deep-interview · office-hours

“What else could it be?” Enumerate siblings. Compare alternatives. Never commit. The session ends with ten options and zero roots.

deeper

“Why is this really true?” Commit to one claim. Descend. Never widen until it hits bedrock — a stated value, a constraint, a prior decision, an external rule, a measured fact.

One drill · many rounds

Cold context every round.
The cursor is always the deepest open leaf.

It runs as a Claude Code dynamic workflow. Each round fans out cold subagents in parallel, a judge keeps the deepest answer, and the cursor descends — looping until every leaf reaches a verified bedrock.

Find

Locate the cursor — the DFS-deepest open leaf. Build its ancestor chain: root → … → claim. Never siblings, never the whole tree.

Fan out

Spawn N cold subagents in parallel on the same claim. Fresh context each — no rationalizing prior reasoning into self-consistency.

Judge

A judge keeps the single deepest, best-typed answer: descend, branch, or declare bedrock. Schema-validated — prose-buried endings can’t slip through.

Descend

Move the cursor down. Close leaves at bedrock; pop to the next open one. Done when the tree says so — not when the model feels finished.

Four commitments

What keeps it honest.

01 — ralph

Cold re-injection

In a long session, context accumulates and adversarial pressure decays into self-consistency. deeper re-injects fresh context every round — only the prompt, lessons, and ancestor chain enter.

02 — DFS

Depth-first to bedrock

Drill one claim straight down. A normal answer descends; a bedrock declaration closes the leaf; a branch opens a parallel cause. Never “what else?”

03 — termination

Disciplined stop

Bedrock is an axiom — a value, constraint, prior decision, rule, identity, or measured fact. Not a turn count. Shallow bedrock is a flagged violation.

04 — flywheel

Self-improvement

Each drill records its rule violations; recurring ones become binding lessons. The next drill loads them and biases away from the repeated mistake — no one edits the prompt by hand.
